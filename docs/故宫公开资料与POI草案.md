# 故宫公开资料与第一版总图、POI 草案

## 结论

这版草案优先使用故宫博物院官方公开资料整理，适合先落到当前项目中做第一版真实数据接入。

当前最有价值的公开来源有三类：

1. 官方导览页
2. 官方开放区域总图
3. 官方讲解服务路线图

## 公开资料来源

### 1. 官方导览页

- 来源：[故宫博物院导览页](https://www.dpm.org.cn/Visit.html)
- 可提取信息：
  - 单向游览规则
  - 午门入口、神武门和东华门出口
  - 开放宫殿区名单
  - 常设专馆名单
  - 服务设施种类

关键信息：

- 故宫博物院实行自南向北单向参观，观众由午门进入，参观结束后可由神武门或东华门离开。
- 官方页面列出了开放宫殿区，包括：
  - 午门、太和门、太和殿、中和殿、保和殿
  - 乾清宫、交泰殿、坤宁宫
  - 文华殿、文渊阁、武英殿
  - 养心殿区、斋宫区、奉先殿区
  - 东六宫、西六宫、御花园、宁寿宫区
- 官方页面列出了常设专馆，包括：
  - 书画馆
  - 陶瓷馆
  - 珍宝馆
  - 钟表馆
  - 雕塑馆
  - 戏曲馆
  - 石鼓馆
  - 家具馆

### 2. 官方开放区域总图

- 来源页：[导览页中的“故宫博物院开放区域”](https://www.dpm.org.cn/Visit.html)
- 直接图片地址：[官方开放区域总图](https://img.dpm.org.cn/static/themes/image/xf/map3.jpg)

用途建议：

- 这张图可以直接作为第一版总图参考底图
- 更适合“内部研究、原型定位、坐标标定”
- 如果要正式上线或公开发布，建议确认图片授权使用范围，或改为你自有授权底图

### 3. 官方讲解服务路线图

- 来源页：[故宫博物院讲解服务路线与收费标准](https://www.dpm.org.cn/announce_detail/239321.html)
- 路线图图片：
  - [精品小团路线图](https://img.dpm.org.cn/Uploads/image/2024/11/01/1-CHnSwXlEi260.jpg)

- 中轴线 + 西六宫 + 珍宝馆

- 中轴线 + 西六宫 + 慈宁宫区域

### 4. 官方暑期参观提示

  - 文创
  - 书店
  - 卫生间

## 第一版总图建议

### 可直接使用的第一版方案

- 底图来源：官方开放区域总图
- 接入方式：
  - 先作为参考底图标定坐标
  - 再根据需要替换成授权更清晰的总图

### 我建议的处理方式

1. 先用官方总图把 POI 坐标和路线坐标跑通
2. 再替换成你后续拿到的高清授权总图
3. 替换后仅需二次校准 `x_ratio` 和 `y_ratio`

- 服务点在图上的实际位置
- 当前期专题展名称与时间
